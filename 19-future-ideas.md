# Future Ideas

These are intentionally not part of MVP.

## Timed Challenges

Future versions could support a timed nudge.

Examples:

- morning: start gently
- afternoon: reset attention
- evening: close the loop

Keep one challenge per day unless testing proves otherwise.

## More Present Reminders

Potential future behavior:

- stronger notification copy
- menu bar icon pulse
- reminder after long inactivity
- reminder during common slump hours

Guardrail:
Do not become annoying.

## Smart Notification Timing

Possible logic:

- if incomplete by 11:00 AM, remind lightly
- if incomplete by 2:00 PM, send main reminder
- never send after user completes
- never send more than once or twice per day

Reminders are not part of the current MVP.

## Menu Bar Visual Pressure

Future visual nudges:

- incomplete icon becomes slightly more noticeable after reminder time
- filled square animation when complete
- heatmap preview becomes more visually satisfying

## Challenge Packs

Possible packs:

- work-from-home
- screen break
- body reset
- desk reset
- calm mode

Do not build packs until the core loop proves sticky.

## Reroll

A future version may allow one reroll per day.

Rules:

- only one reroll
- no endless browsing
- reroll is not the main loop

## Weekly Reflection

Optional future feature:

- simple weekly view
- no productivity analytics
- just count completed squares

Avoid dashboards unless users ask for them.
